# IntContainer

Простой контейнер для хранения произвольного количества целых чисел, реализованный на Java без использования встроенных коллекций.

Проект создан в учебных целях для демонстрации работы с:
- пользовательскими структурами данных
- системой сборки Maven
- системой контроля версий Git

---

## Возможности контейнера

Контейнер поддерживает следующие операции:

- добавление элементов
- получение элемента по индексу
- удаление элемента
- получение количества элементов

Контейнер реализован на основе **динамического массива**, который автоматически увеличивает размер при переполнении.
